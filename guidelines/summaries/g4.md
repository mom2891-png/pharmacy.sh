# 2023 고혈압 진료지침 전문 요약 (KSH Clinical Practice Master)

## 1. 질환의 정의 및 정밀 진단 (Definition & Diagnosis)
정확한 혈압 분류는 치료 시작 시점과 약제 선택의 기준이 됩니다.

| 분류 (Category) | 진료실 혈압 (Office) | 가정 혈압 (HBPM) | 활동 혈압 (주간 평균) |
| :--- | :---: | :---: | :---: |
| **정상 혈압** | $< 120 / 80$ | $< 115 / 75$ | $< 115 / 75$ |
| **주의 혈압** | $120$-$129 / < 80$ | - | - |
| **고혈압 전단계** | $130$-$139 / 80$-$89$ | $130$-$134 / 80$-$84$ | $130$-$134 / 80$-$84$ |
| **1기 고혈압** | $140$-$159 / 90$-$99$ | $\ge 135 / 85$ | $\ge 135 / 85$ |
| **2기 고혈압** | $\ge 160 / 100$ | $\ge 145 / 92$ | $\ge 145 / 92$ |

## 2. 치료 목표 및 결과 지표 (Therapeutic Goals & Outcomes)
단순한 수치 강하를 넘어 심뇌혈관 사건(MACE) 및 표적 장기 손상 예방을 목표로 합니다.

*   **일반 고혈압 환자 (저/중위험군)**: **$< 140/90$ mmHg** (기본 목표).
*   **적극적 조절군 (고위험군)**: **$< 130/80$ mmHg**.
    *   **당뇨병**: 표적 장기 손상이 있거나 심혈관 위험 인자가 동반된 경우.
    *   **만성 콩팥병(CKD)**: 알부민뇨가 동반된 경우(UACR $\ge 300$ mg/g).
    *   **심혈관 질환**: 관상동맥질환, 뇌졸중(열공성 뇌경색) 등 기왕력자.
*   **노인 고혈압 ($> 65$세)**: 수축기 혈압 **$< 140$ mmHg** 목표. (내약성이 좋은 경우 $130$ mmHg 미만까지 고려 가능하나, 확산기 혈압이 $60$ mmHg 미만으로 떨어지지 않도록 세밀한 모니터링 필요).

## 3. 심층 약물 요법 및 치료 전략 (Advanced Pharmacotherapy Strategy)

### **① 주요 약제군별 약리학적 심층 분석 (Drug Class & Pharmacology)**

*   **RAS 차단제 (ACEi / ARB)**: 
    *   **ACE 억제제 (Ramipril, Enalapril)**: Bradykinin 분해 억제로 심보호 효과가 우수하나 한국인 $20$–$30$%에서 마른 기침 발생.
    *   **ARB (Losartan, Valsartan, Telmisartan, Fimasartan)**: AT1 수용체를 직접 차단. **Telmisartan**은 긴 반감기로 야간 혈압 조절에 유리하며, **Fimasartan**은 강력한 수용체 결합력으로 강력한 강압 효과를 보임.
    *   **Pearl**: 사구체 내압 감소를 통한 신장 보호 효과가 탁월하여 당뇨병 환자에게 1차 선택됨.

*   **칼슘통합차단제 (CCB - DHP계)**: 
    *   **성분**: **Amlodipine, Lercanidipine, Nifedipine (GITS)**.
    *   **기전**: L-type 칼슘 채널 차단으로 세동맥 확장.
    *   **특징**: 대사적 중립성(혈당/지질 영향 없음)이 좋아 노인 및 대사증후군 환자에게 선호됨. **Lercanidipine**은 발목 부종 부작용이 상대적으로 적음.

*   **Thiazide계 이뇨제**: 
    *   **성분**: **Chlorthalidone, Indapamide**, HCTZ.
    *   **기전**: 원위세뇨관 나트륨 재흡수 억제. **Indapamide**는 직접적인 혈관 확장 효과를 병행하여 심혈관 예방 데이터가 풍부함.

### **② 단계별 치료 및 Titration 로직**
1.  **Step 1 (초기)**: 대부분 저용량 **2제 병용 요법(SPC)**으로 시작. (강력한 초기 조절이 장기 예후에 유리함)
2.  **Step 2 (적정)**: 서로 다른 기전의 약물을 표준 용량(Full-dose)으로 조합.
3.  **Step 3 (3제)**: **ARB + CCB + Diuretic** 조합. 이뇨제는 CCB에 의한 부종을 완화하는 시너지를 가짐.

## 4. 약사 전문 복약 중재 및 안전성 (Safety & Monitoring)
*   **The Triple Whammy**: **RAS 차단제 + 이뇨제 + NSAIDs** 병용 시 신장 관류압 저하로 인한 **급성 신부전(AKI)** 위험. 약국 스크리닝 필수.
*   **전해질 관리**: 이뇨제 사용 시 저칼륨혈증, ARB/ACEi 사용 시 고칼륨혈증 모니터링.
*   **DDI**: CCB와 자몽 주스(CYP3A4 억제) 병용 시 혈중 농도 상승으로 인한 저혈압/두통 주의.

| 동반 질환 | 추천 (Best Option) | 주의/금기 | 근거 |
| :--- | :--- | :--- | :--- |
| **심부전 (HFrEF)** | ACEi/ARB, BB, MRA, SGLT2i | Non-DHP CCB | 심수축력 저하 방지 |
| **당뇨/단백뇨** | **ACEi / ARB** | - | 사구체 내압 감소 및 신보호 |
| **협심증** | BB, CCB | - | 심근 산소 요구량 감소 |
| **천식/COPD** | CCB, ARB | 비선택적 BB | 기관지 수축 유발 방지 |
