# 일차 의료용 성인 천식 Quick Reference Guide

## 1. 진단 및 스크리닝 (Rapid Screening)
*   **천식 의심**: 가변적인 호흡곤란, 천명, 가슴 답답함, 기침.
*   **확진**: $FEV_1$ 가변성 증명 ($12\%$ & $200$mL 증가).

---

## 2. 치료 목표 및 조절도 평가 (Quick Target)
*   **조절 목표**: 증상 없음, 활동 제한 없음, 완화제 사용 최소화.
*   **조절도 평가 (최근 4주)**:
    | 항목 | 잘 조절됨 | 부분 조절 | 조절 안 됨 |
    | :--- | :---: | :---: | :---: |
    | 주간 증상 ($>2$회/주) | 없음 | 1~2개 해당 | 3~4개 해당 |
    | 야간 각성 | 없음 | | |
    | 완화제 사용 ($>2$회/주) | 없음 | | |
    | 활동 제한 | 없음 | | |

---

## 3. 핵심 약물 요법 및 알고리즘 (Rapid Algorithm)

### **① 주요 흡입제 계열 및 성분 비교**
| 계열 | 성분 예시 (Brand) | 특징 및 Insight |
| :--- | :--- | :--- |
| **ICS (조절제)** | Budesonide (풀미코트), Fluticasone (플루티폼 성분) | 항염증의 근간. 단독 유지요법 가능. |
| **ICS-LABA (복합제)** | Budesonide/Formoterol (심비코트), Fluticasone/Salmeterol (세레타이드) | 조절 및 유지. **심비코트**는 완화제로도 사용 가능(MART). |
| **SABA (완화제)** | Salbutamol (벤토린) | 급성 증상 완화. 규칙적 사용은 금기. |
| **LAMA (추가제)** | Tiotropium (스피리바) | 4-5단계 추가 약제. 폐기능 개선 효과. |

### **② GINA Track 1 선호 단계 (MART 전략)**
*   **Step 1–2**: 증상 시에만 저용량 **ICS-Formoterol**.
*   **Step 3**: 저용량 **ICS-Formoterol** 매일 유지 + 필요시 사용.
*   **Step 4**: 중등도 **ICS-Formoterol** 매일 유지 + 필요시 사용.

---

## 4. 약사 전용 안전성 체크 (Safety Red Flags)
*   **SABA 오남용**: 한 달에 벤토린 1개 이상 사용 시 **천식 악화 사망 위험** 급증 경고.
*   **입 헹구기 교육**: "가글하지 않으면 입안에 곰팡이가 생기거나 목소리가 쉴 수 있습니다."
*   **흡입기 잔량 확인**: 카운터가 없는 구형 모델의 경우 물에 띄우는 방식 등은 정확하지 않으므로 사용 횟수 기록 권장.

---

## 5. 의뢰가 필요한 Red Flags (Referral Guide)
*   **FEV1 < 60%** 인 경우.
*   연간 2회 이상의 급성 악화.
*   약물 순응도가 좋음에도 조절되지 않는 중증 천식.
*   직업성 천식 의심 (직장에서 증상 악화).
